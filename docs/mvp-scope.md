# MVP Scope

## Included in MVP

- process creation
- process list
- process detail
- structured process intake
- role input
- task input
- dependency input
- simple decision point input
- BPMN draft generation from structured data
- BPMN display
- BPMN basic editing
- save/reload
- owner/status/notes metadata

## Explicitly Excluded from MVP

- full AI generation from only messy meeting transcripts
- real-time collaboration
- enterprise-grade permissions
- advanced workflow execution engine
- advanced BPMN event coverage
- analytics dashboards beyond simple summaries
- external integrations
- advanced template marketplace
- complex version branching

## Product Discipline Rule

If a proposed feature does not directly improve the MVP ability to:

- capture a process
- generate a BPMN draft
- review/edit that BPMN
- maintain planning metadata

then it is probably not MVP-critical.
