# Design System

## Brand Direction

The product should visually align with the existing logo and brand identity:

- dark
- premium
- corporate
- controlled
- technical
- professional

Avoid playful startup UI language.

## Core Visual Principles

- black owns the space
- gold signals importance
- gray stabilizes structure
- whitespace creates premium feel
- surfaces must remain distinct and readable

## Typography

Primary UI font:
- Inter

Use:
- strong, clean headings
- readable body text
- restrained letter spacing
- occasional uppercase labels for premium feel

## Radius

Recommended:
- 12px default
- 16px for large cards / modals

## Border and Shadow

- borders should be subtle
- shadows should be minimal
- avoid glowing or neon styling

## Buttons

### Primary
Brand gold background, dark text if readable or light text depending on contrast.

### Secondary
Dark surface with border.

### Ghost
Minimal, text-based, subtle hover.

## Inputs

- dark surface
- subtle border
- clear focus state
- premium spacing
- readable contrast

## BPMN Canvas Styling

- canvas should not blend fully into app background
- use a slightly distinct surface
- keep nodes readable
- use gold for focus/selection, not everywhere

## Motion

Use minimal motion only:
- panel transitions
- hover transitions
- subtle fades
- clean focus states

Avoid:
- bounce
- neon glow
- excessive pulse
- flashy demo animations
